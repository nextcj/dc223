# 视频指导：升级

### 一. 升级应用程序

<pre>
1. 连接 WIFI 网络
2. 进入 [系统] 菜单
3. 下拉看到 [应用更新] 选项，点击它
4. 如果有新版本就会弹窗正在下载的进度
5. 等待下载完成，按照提示进行安装
</pre>

### 二. 升级屏幕安卓系统

<pre>
1. 接上 U 盘
2. 进入 [系统] 菜单
3. 进入第一个选项 [apps]
4. 点击 [OTA 升级]
5. 选择第一个选项 [Check]
</pre>
